# Reference Photos

Place real photos here for AI generation consistency. These images will be used as references for LoRA training, IP-Adapter, PuLID, and FLUX Kontext workflows.

## Folder Structure

```
reference-photos/
├── motorcycle/     ← Kawasaki Ninja H2 photos (angles, details, colors)
├── car/            ← Car photos (angles, interior, details)
├── helmet/         ← AGV Pista GP RR helmet photos
├── outfits/        ← Clothing reference (leather jacket, lingerie, casual)
├── environments/   ← Location references (garage, bedroom, bathroom, roads)
└── README.md
```

## Tips for Reference Photos

- **Multiple angles**: Front, side, 3/4, rear, detail shots
- **Good lighting**: Clear, well-lit photos work best
- **High resolution**: Minimum 1024px on the longest side
- **Clean background**: Helps AI isolate the subject
- **Real photos preferred**: AI models work better with real photo references than renders
